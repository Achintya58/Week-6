This breadboard will be used to receive inputs of the Safe Combination.

If the Safe Opening procedure is done correctly, the Huzzah board will receive power and transmit it to a button. If the correct buttons are pressed in the correct order, the PWMpin at the end of the buttons will sense a "HIGH", and therefore start Reading the 3rd part of the Circuit, or the ones with the Infrared LED and Phototransistor. If the PWMpin originating from the Phototransistor reads a "HIGH", the Circuit will start reading an Accelerometer Breadboard (not shown). If the Accelerometer is held correctly, the Code will instruct the Motors connected to the Circuit, as well as Certain LEDs to light up, therefore opening the safe.
